Restroom Alert
==============

Uses a Spark core
![](https://www.adafruit.com/images/1200x900/2127-00.jpg)

### Install
npm install

### Runserver

$ node app.js

### Info

Monitor a door status using spark core and sensors

This web application queries the Spark Core API for data and updates the DOM based on its results.
